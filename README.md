# Dashboard IT

Dashboard IT personnel développé en Python avec PySide6.

## Lancer le projet

```bash
python app.py
```

### Fonctionnalités
 - Interface graphique dark
 - Gestion de tâches (SQLite)
 - Dashboard d’accueil avec indicateurs
 - Recherche et filtres

### Prérequis
 - Windows 10 / 11
 - Python 3.12+
 - Environnement virtuel Python activé
 - PySide6, qtawesome
